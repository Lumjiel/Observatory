# 《编码》全书笔记

作者 Charles Petzold，这本书从最基础的手电筒开始，逐步构建出完整的计算机系统。

## 第一部分：编码与组合

### 手电筒的秘密
- 信息可以用开/关两种状态表示，这就是编码
- 摩尔斯电码证明了二进制编码可以表达复杂信息

### 继电器与逻辑门
- 继电器是一种电磁开关
- 多个继电器组合可以实现 AND、OR、NOT 门
- 这些门构成了计算机的基础

```
继电器 -> 门 -> 触发器 -> 计数器 -> 存储器
```

## 第二部分：处理器

### 加法器的构建
- 半加器：两个比特相加，输出和与进位
- 全加器：考虑来自低位的进位
- 8位加法器串联多个全加器

### 减法如何实现
- 补码表示法使减法变为加法
- 溢出处理是关键考量

## 第三部分：存储器

### 触发器与锁存器
- D触发器：时钟上升沿采样
- 寄存器：多个触发器组成

### 随机存取存储器
- 256x8 RAM 阵列的构建过程
- 地址线、数据线、控制线的配合

## 读后感

整本书贯穿一个核心思想：**复杂系统是从简单组件层层叠加构建的**。
